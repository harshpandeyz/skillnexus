

# 🧭 SkillNexus — Responsive Learning Platform

Personal, responsive learning platform built with **HTML5**, **CSS3**, **JavaScript** and **Bootstrap**.  
A student project that demonstrates responsive UI, forms, client-side cart simulation and basic demo authentication.

**Open the project (local):** [`./src/index.html`](./src/index.html) — or host the `src` folder with GitHub Pages / Vercel / Netlify.

### 🎯 Key Features
- ✅ Responsive layout for desktop/tablet/mobile (Bootstrap + custom CSS)  
- 🧭 Multi-page app: **Home, About, Contact, Cart, Register, Login**  
- 🖼️ Custom right→left advertising slider on Home page  
- 🛒 Client-side cart using `localStorage` (add / qty / remove / checkout demo)  
- 📝 Contact form with multiple input types (saved in `localStorage`)  
- 🔐 Register & Login pages (demo authentication stored in `localStorage`)  
- 🧩 Clean, commented code separated into `style.css` and `script.js`
